Cara menjalankan Notebook.

1. Google Colab:
    1. Buka Google Colab
    2. Upload file notebook (.ipynb) melalui menu File → Upload notebook.
    3. Klik Runtime → Run all untuk menjalankan seluruh sel.

2. Lokal (Jupyter Notebook)
     1. Install Python terlebih dahulu.
     2. Install Jupyter dan library yang dibutuhkan:
        (pip install jupyter pandas numpy matplotlib seaborn scipy scikit-learn)
     3. Buka terminal pada folder notebook.
     4. Jalankan: jupyter notebook
     5. Browser akan terbuka secara otomatis.
     6. Pilih file .ipynb dan klik Run All Cells.
